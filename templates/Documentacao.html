! Documentação da API Scraping !

Para acessar a api corretamente deve se informar os argumentos obrigatorios da API

Endpoint: /api/solicitar/

Argumentos: quantidade | produto


Exemplo de url com os argumentos :
/api.solicitar/?produto={Produto}&quantidade={quantidade}


Os produtos serao usados para consulta e a quantidade e define quantos produtos serao retornados, automaticamente retornara o produto mais barato e todas as respostas sao em JSON



Exemplo de Resposta 



"Monitor Fullhd Pctop Led 19 60hz Hdmi Vga Mlp190hdmi  Barato",
                   
{"url": "https://produto.mercadolivre.com.br/{url do produto}",
"valor": 86.25,
"vendedor": "Mercado Livre"}
